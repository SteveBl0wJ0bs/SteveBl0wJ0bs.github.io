In this histogram, you can see how the size of the trees is distributed. The maximum height is equal to 60 (x-axis). Click on the button under the histogram to choose the number of bins and to see how the distribution change. In some cases you might not see all the bins you selected using the bottom. This phenomenon is due to the fact that very few trees has a certain height.


Now through a boxplot, we represent the same previous context. To ensure a good visualization of the boxplot, we consider only the six most present species.


In this scatterplot, you can see the distribution of CO2 stored respect to the height of trees belonging to the top six most present species since it's unfeasible to assign to each one of 224 categories a good different color. In particular, the most of trees with an height less than 25m has CO2 values almost equal to the half of the maximum value. The dots with the same color are of the same specie. Click on the dots to see the name specie's name and the exact values of CO2 and height.


The following one is a multiple scatterplot showing the same data of the previous one but using one separated plot for each different plant among the top 6 most present plants.
Put the cursor over a point to see the name of the plant associated with that subgraph.

Now through a bubble chart, we represent the same previous context, but this time the canopy size is also considered: higher is the canopy size, bigger are the dots. The dots with the same color are of the same specie.
Click on the dots to see the name specie's name and the exact values of CO2, height and canopy size.
